# Smart Road Safety System Using Driver Drowsiness Detection

## Overview
A smart road safety system that detects driver drowsiness and alerts the driver to prevent accidents.

## Objectives
- Detect driver fatigue in real time
- Alert the driver before accidents occur
- Improve road safety
- Provide a low-cost solution

## Components
- Arduino Uno
- IR Sensor
- Buzzer
- DC Motor

## Features
- Real-time monitoring
- Automatic alert system
- Accident prevention
- Low-cost implementation
